# Cancer Patients & Air Pollution Classification App

This Streamlit app uses a Random Forest model to predict cancer risk
based on patient attributes and lifestyle/environment factors.

## Instructions

1. Install requirements:
    ```bash
    pip install -r requirements.txt
    ```

2. Run Streamlit app:
    ```bash
    streamlit run app.py
    ```

## Deployment

Deploy via Streamlit Cloud by linking this GitHub repo.
